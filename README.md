# TailwindLandingPages 🚀

## Overview
A collection of modern, responsive landing page prototypes built with **HTML** and **Tailwind CSS**. Perfect for quick demos, portfolios, or starting points for web projects.

## Tech Stack
- **Tailwind CSS** v3.4+
- **HTML5**
- **Images/Assets** in `src/images/` & `src/pics/`

## Quick Start 🛫
1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Development (watch CSS):**
   ```bash
   npm run dev
   ```
   - Compiles `src/input.css` → `src/output.css`
3. **Preview sites:**
   ```bash
   npx live-server src/
   ```
   Or open `src/*.html` directly in browser.
